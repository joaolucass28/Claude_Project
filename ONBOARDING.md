## Your Setup Checklist

### Codebases
- [ ] claude_project — github.com/joaolucass28/claude_project

### MCP Servers to Activate
- [ ] **Framer MCP** — connects Claude directly to your Framer projects so you can read/write pages, CMS collections, code files, and components without leaving the terminal. Ask financebro for the connection details in Claude's MCP settings.

### Skills to Know About
- `/agents` — view and interact with specialized sub-agents (email drafter, website reviewer, website strategy, brand-dna, etc.). Used constantly here for client work.
- `/btw` — quick side note or context injection mid-session. Useful when you remember something relevant without breaking flow.
- `/clear` — clears the conversation context. Use between unrelated tasks to keep Claude focused.
- `/effort` — controls how much reasoning Claude applies. Dial up for complex strategy, dial down for quick lookups.
- `@email-draft-agent` — drafts client emails, shows you subject + body, waits for approval before touching Mail.app. Never sends automatically.
- `@website-strategy` — structured website strategy workflow: positioning, copy, sitemap, visual direction, Framer notes.
- `@website-qa-reviewer` — pre-delivery site audit. Run before handing off any client site.
