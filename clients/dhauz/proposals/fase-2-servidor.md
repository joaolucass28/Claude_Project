# Proposta Fase 2 — Servidor Dhauz *(planejada)*

**Status:** Planejada — iniciar após Fase 1 estabilizada (3–6 meses)  
**Pré-requisito:** Fase 1 concluída e rede operando de forma estável

---

## Objetivo

Transformar a infraestrutura de rede da Dhauz em uma central de operação digital: armazenamento centralizado, aplicações internas, automações e base para escalar a operação.

---

## Sub-fases

### Fase 2A — NAS (ponto de entrada)

**O que é:** Servidor de armazenamento e backup — a camada mais simples antes do servidor real.

| Entrega | Descrição |
|---------|-----------|
| NAS configurado | Armazenamento central acessível por toda a rede |
| Backup automático | Rotina de backup local + cloud |
| Compartilhamento interno | Pastas por equipe, acesso controlado |
| Acesso remoto | Acesso seguro de fora da empresa |

**Investimento estimado:** R$ 3.000 – 8.000 (NAS + HDs)  
**Prazo:** 1–2 semanas após aprovação

---

### Fase 2B — Servidor real

**O que é:** Infraestrutura para rodar aplicações, banco de dados e automações internas.

| Entrega | Descrição |
|---------|-----------|
| Servidor dedicado | Hardware local para aplicações |
| Banco de dados | Estrutura para dados internos da operação |
| Automações | Scripts e rotinas automatizadas |
| Controle de mídia | Armazenamento e gerenciamento de arquivos pesados |

**Investimento estimado:** R$ 8.000 – 15.000  
**Prazo:** 2–4 semanas

---

### Fase 2C — Infraestrutura completa

**O que é:** Operação digital completa — IP fixo, fibra robusta, servidor escalável.

| Item | Estimativa |
|------|-----------|
| Servidor dedicado ou workstation | R$ 6.000 – 15.000 |
| Armazenamento SSD + HD híbrido | R$ 2.000 – 6.000 |
| Backup local + cloud | R$ 500 – 2.000 |
| Upgrade de internet (fibra + IP fixo) | R$ 200 – 600/mês |
| **Total** | **R$ 15.000 – 25.000+** |

---

---

## Notas

- Não há prazo ou pressão para iniciar — depende da maturidade da operação pós-Fase 1
- A Fase 1 foi projetada para não criar retrabalho quando a Fase 2 começar
- Cada sub-fase pode ser aprovada e executada de forma independente
